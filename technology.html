03 · Activity Index

테마별 활동지수 알고리즘

기상·해양 요소를 레저스포츠 종목별 가중치로 결합해, 오늘 바다에 나가도 되는지를 하나의 지수로 답합니다.

Activity Index (AI) = W₁·S₁ + W₂·S₂ + W₃·S₃ + ⋯ + Wₙ·Sₙ

Sₙ — 각 요소의 점수 (0–10) · 레저스포츠 종별 변수 범위에 따른 점수 산정
Wₙ — 각 요소의 가중치 · 레저스포츠 종목별 조정
